# Smart Crop Rotatory Advisor

## Overview
Smart Crop Rotatory Advisor is a statistical analysis project focused on understanding how soil properties and environmental factors affect crop yield. The project analyzes agricultural datasets to identify patterns and relationships that can help optimize crop rotation strategies and improve farming productivity.

## Objectives
- Analyze soil and crop datasets.
- Identify factors that significantly impact crop yield.
- Study the influence of soil type, nutrients, rainfall, and seasonal conditions.
- Generate insights for effective crop rotation planning.
- Support sustainable agricultural practices through data-driven decisions.

## Dataset Information
Dataset: soil.impact.csv

### Dataset Summary
- Number of Rows: 15,400
- Number of Columns: 15
- Numerical Features:
  - Temperature
  - Rainfall
  - pH
  - Light Hours
  - Light Intensity
  - Relative Humidity
  - Nitrogen
  - Phosphorus
  - Potassium
  - Yield

- Categorical Features:
  - Crop Name
  - Fertility
  - Soil Type
  - Season
  - Impact

## Key Findings
- Loam soil consistently produces higher average crop yields.
- Restorative crops generally achieve better yields compared to neutral and depleting crops.
- The dataset is balanced across all four seasons, reducing seasonal bias.
- Crop yield distribution is right-skewed, with most yields ranging between 10 and 30 units.
