# Smart Crop Rotatory Advisor

Smart Crop Rotatory Advisor is a Python-based project that analyzes soil and crop data to identify factors affecting yield and support smarter crop rotation decisions.

## Features
- Soil analysis
- Crop yield analysis
- Statistical visualization
- Crop rotation insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
soil.impact.csv
